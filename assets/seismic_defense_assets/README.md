# 내진 디펜스 게임 에셋 모음

이 압축 파일은 제공된 픽셀 아트 스프라이트 시트를 개별 PNG 파일로 잘라 정리한 것입니다.

## 폴더 구성
- 00_minimum_required: 바로 코드에 연결하기 좋은 최소 필수 에셋 12개
- 01_building_tiles: 건물 타일, 벽, 창문, 기초, 바닥 타일
- 02_device_icons: X자 가새, 전단벽, 제진 장치, 면진 장치, TMD, 철거 도구 등
- 03_effects: 지진파, 먼지, 파편, 균열, 경고, 성공/보강 배너
- 04_npcs: 안전요원, 시장님, 주민 캐릭터 표정별 스프라이트
- 05_environment: 도시 배경, 크레인, 굴착기, 울타리, 콘, 구름, 땅
- 06_ui_components: UI 프레임, 버튼, 카드 등 자동 분리 파일
- 07_badges: 성취 배지 아이콘
- 08_stage_map: 스테이지 맵 아이콘과 연결선
- original_sheets: 원본 스프라이트 시트

## 참고
- 배경 체크무늬는 자동으로 투명 처리했습니다. 일부 가장자리는 수동 보정이 필요할 수 있습니다.
- 파일별 원본 위치는 manifest.json에 기록했습니다.
